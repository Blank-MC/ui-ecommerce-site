## ðŸš€ Memulai Proyek (Get Started)

Ikuti langkah-langkah di bawah ini untuk menjalankan salinan lokal proyek ini.

### Prasyarat

Anda hanya memerlukan peramban web modern (Google Chrome, Firefox, dll.).

### Instalasi

1.  **Kloning Repositori:**
    ```bash
    git clone https://github.com/Blank-MC/ui-ecommerce-site.git
    ```

2.  **Masuk ke Direktori Proyek:**
    ```bash
    cd ui-ecommerce-site
    ```

3.  **Jalankan Proyek:**
    Buka file `index.html` langsung di peramban web Anda.

    > **Tips Profesional:** Gunakan ekstensi *Live Server* di VS Code untuk me-reload halaman secara otomatis setiap kali Anda menyimpan perubahan.
